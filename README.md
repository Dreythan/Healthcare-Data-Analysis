# Healthcare Data Analysis Project

## Overview
This project focuses on cleaning and analyzing healthcare data using Python. The goal is to transform raw data into meaningful insights through data preprocessing and visualization.

## Key Features
- Data cleaning and preprocessing
- Feature engineering (length of stay)
- Handling missing values
- Exploratory Data Analysis (EDA)
- Data visualization using matplotlib

## Tools Used
- Python
- pandas
- matplotlib
- Jupyter Notebook

## Insights
- Identified trends in patient length of stay
- Analyzed billing distributions across conditions
- Observed variations in healthcare costs by medical condition

## Files
- healthcare_analysis.ipynb → main analysis notebook
- healthcare_dataset.csv → raw dataset
- healthcare_dataset_clean.csv → cleaned dataset
